# 小型エアコン

ペルチェ素子（Peltier element）を活用した、卓上サイズの小型エアコン開発プロジェクトです。

---

## 基本的な仕様

本プロジェクトで開発する小型エアコンの基本コンセプトと仕様案です。

### 1. 冷却・加熱メカニズム
* **熱電変換技術:** ペルチェ素子（TEC1-12706などを想定）に電流を流すことで生じる熱移動を利用。
* **排熱システム:** 素子の高温面に大型ヒートシンクと冷却ファンを組み合わせ、効率的に排熱。
* **冷気拡散:** 低温面に小型ヒートシンクを配置し、ファンで冷気を前方に送り出す。

### 2. 想定スペック（暫定）
* **サイズ:** デスクに置けるコンパクトサイズ
* **電源:** DC 12V（ペルチェ素子およびファンの駆動用）
* **制御（今後追加予定）:** マイコン（ArduinoやESP32など）を用いた温度管理や風量調節
